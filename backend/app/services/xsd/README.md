# SAT XSD Schemas

Place official SAT CFDI XSD files here for XML validation.

Required files:
- `cfdv40.xsd` — CFDI 4.0 (current standard)
- `cfdv33.xsd` — CFDI 3.3 (legacy, still in circulation)

Download from SAT:
- https://www.sat.gob.mx/sitio_internet/cfd/4/cfdv40.xsd
- https://www.sat.gob.mx/sitio_internet/cfd/3/cfdv33.xsd

These files are NOT included in the repository (SAT intellectual property).
Without them, XSD validation is skipped but parsing still works.
